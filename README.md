## Create new app

## Adding bootstrap 
npm install --save bootstrap@5.3.0-alpha1

Go to angular.json file to the section build->styles
 "styles": [
              "src/styles.css",
              "node_modules/bootstrap/dist/css/bootstrap.css"
            ],

## Creating components
